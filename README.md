# 🌸 Soins des cheveux crépus sous le voile

## 📌 Description du projet
Ce site a pour objectif de donner des conseils simples et pratiques pour entretenir les cheveux crépus sous le voile.  
Il propose des astuces naturelles et des routines capillaires pour garder des cheveux en bonne santé, hydratés et protégés au quotidien.

---

## 🎯 Objectif
Aider les personnes à mieux comprendre comment prendre soin de leurs cheveux crépus, même sous le voile, afin de préserver leur beauté et leur santé.
